# Guestbook
 It's a simple guestbook site on golang. You can see comment another users and add your comment.
 
 This project use:
 - Json
 - Http
 - Docker
 - Adaptive layout 

### Docker commands: 

 ```ruby
docker build -t guestbook .
```
```ruby
docker run -it -p 8080:8080 guestbook
```
### Result your link - http://localhost:8080/guestbook
